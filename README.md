# Interpol (interpol)
INTERPOL (International Criminal Police Organization) is an inter-governmental organization with 196 member countries that helps police worldwide work together to make the world a safer place. INTERPOL provides an API for accessing Red, Yellow and UN Notices data.

## APIs

### Interpol Notices API
The Interpol Notices API provides programmatic access to INTERPOL Red, Yellow and UN Notices, enabling law enforcement and authorized organizations to query notice information.
